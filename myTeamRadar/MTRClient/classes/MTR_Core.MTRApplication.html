Start

  • Start(url?: string, param?: unknown): void

  • Starts the Application.

    Description

    If 'url' is passed, this is assumed to be the endpoint of an API to call to initialise the application. If 'param' is null, the endpoint is called via the GET method, otherwise POST is used passing 'param' as parameters.

    Before calling an endpoint, the routine first checks to see if the data has been saved to a session variable. If not, it calls the routine CallServer passing the routine ProcessJSON as a callback. Eventually, the routine GetCurrentPage is called, either directly from the routine or from ProcessJSON.

    Parameters

    • url: string = null

      URL of API to call to load data.

    • param: unknown = null

      Object containing parameters for API call (by POST).

    Returns void

    Inherited from Application.Start
